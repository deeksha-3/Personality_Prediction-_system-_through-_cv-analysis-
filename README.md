Personality Prediction System Through CV Analysis

A desktop application that predicts personality traits based on user input and uploaded resumes. The system uses Machine Learning (Logistic Regression) and resume parsing to provide insights and generate reports. Built with Python and CustomTkinter for a modern GUI experience.

Features

🧑 User Input  –  Enter personal details and feature responses.
📄 Resume Parsing  –  Upload PDFs or Word documents to extract resume data.
🧠 Machine Learning Prediction  –  Logistic Regression model predicts personality based on input features.
📊 Visualization  –  Displays feature scores in bar charts.
💾 Report Generation  –  Exports a PDF report with the predicted personality.
🖤 Modern GUI  –  Dark-themed interface powered by CustomTkinter.

How It Works

User Inputs  –  Fill in personal details and answer feature-based questions.
Feature Processing  –  Converts inputs into numerical values compatible with the ML model.
ML Prediction  –   Logistic Regression model predicts personality type.
Visualization  –  A bar chart visualizes numeric feature scores.
Resume Parsing  –  Optional resume upload extracts and displays relevant data.
PDF Report  –  Generates a PDF report summarizing the predicted personality.
